## REST API
List of user routes:  

| Route          | HTTP    | Description             |
|----------------|---------|-------------------------|
| /api/users     | GET     | get all users           |
| /api/users:/id | GET     | get specific User       |
| /api/users     | POST    | add user                |
| /api/users/:id | PUT     | edit user specific id   |
| /api/users/:id | DELETE  | delete user specific id |
